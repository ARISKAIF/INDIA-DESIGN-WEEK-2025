## Overview
This repository contains two independent mechanical design projects developed using **Autodesk Fusion 360** as part of **India Design Week 2025**, organized by **Autodesk** in collaboration with **ICT Academy Tamil Nadu**.

The projects focus on **conceptual mechanical design, modular system architecture, material selection, and feasibility-driven engineering**, addressing applications in **multi-environment mobility** and **rehabilitation robotics**.

## Software and Tools
- Autodesk Fusion 360 – Parametric CAD modeling and assemblies

## Project 1: Multi-Mode Hovercraft Vehicle

### Description
A conceptual hybrid vehicle designed for operation across multiple environments:
- Surface travel on water (hovercraft mode)
- Aerial operation using propeller-based lift
- Water propulsion using submerged propeller configurations

The design incorporates **modular and interchangeable propulsion elements**, enabling adaptation to different operating conditions using a common structural platform.

### Engineering Focus
- Multi-domain mobility feasibility  
- Propulsion layout adaptability  
- Structural balance and weight distribution  
- Conceptual hydro-aerodynamic compatibility  

### Scope for Further Development
- Computational fluid dynamics (CFD) analysis  
- Thrust and power estimation  
- Structural and material optimization  
- Control system integration  

---

## Project 2: Rehabilitation Robotic Arm with Linear Rail

### Description
A rehabilitation-oriented robotic arm system mounted on a **linear translational rail**, intended to support controlled and repeatable therapeutic motion in supervised rehabilitation environments.

The design emphasizes **modularity, compactness, and cost awareness**, while maintaining basic kinematic feasibility.

### Technical Summary
- Approximate system mass: 8 kg  
- Estimated cost: ₹80,000 INR  
- Primary structural material: ABS plastic  
- Mounting: Linear rail for extended workspace  
- End-effector: Detachable rehabilitation-specific tool interface  

### Engineering Focus
- Human-centered mechanical design  
- Cost and weight constraints  
- Modular system architecture  
- Basic kinematic feasibility  

### Scope for Further Development
- Load and torque calculations  
- Workspace and motion analysis  
- Structural analysis of joints and rail mount  
- Sensor and feedback integration
