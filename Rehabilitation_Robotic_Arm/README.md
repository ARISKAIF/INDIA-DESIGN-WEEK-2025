# Rehabilitation Robotic Arm with Linear Translation

## Project Overview
This project involves the design of a **rehabilitation-focused robotic arm system** mounted on a **linear translational rail**, intended to assist with controlled and repeatable therapeutic movements.

The design was developed using **Autodesk Fusion 360** as part of **India Design Week 2025**, with emphasis on **cost awareness, modularity, and mechanical feasibility**.

---

## Design Objectives
- Develop a compact and lightweight rehabilitation robotic system  
- Extend operational workspace using linear rail motion  
- Enable task-specific rehabilitation using detachable end-effectors  
- Maintain affordability and ease of fabrication  

---

## System Description

### Robotic Arm Assembly
The robotic arm consists of a multi-joint configuration designed to:
- Support controlled motion suitable for rehabilitation exercises  
- Maintain compact geometry for indoor and clinical environments  
- Allow simplified kinematic motion paths  

The design focuses on **mechanical structure**, without control or actuation optimization.

---

### Linear Rail Mechanism
A linear rail is integrated at the base to:
- Extend the effective workspace of the robotic arm  
- Reduce the need for increased arm reach  
- Support smooth translational motion along a single axis  

---

### Tool Changer and End-Effectors
The system incorporates a **detachable tool interface** allowing:
- Different rehabilitation tools to be mounted  
- Task-specific therapeutic interaction  
- Controlled use under supervised conditions  

Tools are designed exclusively for rehabilitation purposes.
